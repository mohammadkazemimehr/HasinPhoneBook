# HasinPhoneBook

یک Web API ساده برای مدیریت دفترچه تلفن که با **.NET 10** و معماری لایه‌ای مبتنی بر **DDD** توسعه داده شده است. در این سیستم کاربران می‌توانند مخاطبین خود را به همراه چند شماره تلفن و یک تگ (مانند خانواده، کاری یا VIP) ایجاد، ویرایش، حذف و جستجو کنند.

پروژه از چند لایه مجزا تشکیل شده است:

* **Model** برای منطق دامنه، موجودیت‌ها و قوانین کسب‌وکار
* **Application** برای سرویس‌ها، DTOها و مدیریت فرآیندهای برنامه
* **Infrastructure** برای دسترسی به داده‌ها (در حال حاضر به‌صورت In-Memory)
* **API** برای ارائه REST API، مدیریت خطاها و Swagger
* پروژه‌های **Unit Test** و **Integration Test** برای پوشش تست

در مدل Domain، هر مخاطب دارای نام، نام خانوادگی، حداقل یک شماره تلفن و یک تگ اختیاری است. اعتبارسنجی اطلاعات مانند خالی نبودن نام و معتبر بودن شماره تلفن در سطح دامنه انجام می‌شود.

در لایه Application، سرویس‌های مربوط به مدیریت مخاطبین و تگ‌ها پیاده‌سازی شده‌اند. تگ‌ها به‌صورت خودکار در اولین استفاده ایجاد می‌شوند و در دفعات بعدی مجدداً مورد استفاده قرار می‌گیرند.

در حال حاضر داده‌ها در حافظه (In-Memory) نگهداری می‌شوند و با ری‌استارت برنامه از بین می‌روند، اما ساختار پروژه به‌گونه‌ای طراحی شده که بتوان به‌راحتی یک دیتابیس واقعی را جایگزین کرد.

API امکانات کامل CRUD برای مخاطبین و همچنین جستجو بر اساس تگ را فراهم می‌کند و مستندات Swagger نیز در محیط توسعه در دسترس است.
